ID de Usuario El ID de Nombre es obligatorio. Nombre Nombre es obligatorio y solo debe contener letras. Apellido Paterno Apellido Paterno es obligatorio y solo debe contener letras. Apellido Materno Apellido Materno es obligatorio y solo debe contener letras. Teléfono Teléfono debe contener exactamente 10 dígitos. Correo Electrónico Ingresa un correo electrónico válido. Contraseña La contraseña debe tener al menos 8 caracteres, incluir una mayúscula, una minúscula, un número y un carácter especial.
